## 🎨 设计细节

### 1. 章节标题区

**位置**：卡片网格上方居中

**内容**：
```
您的席位包含六大核心权益
SIX CORE ENTITLEMENTS
──────
尊享全方位康养服务，开启品质生活新篇章
```

**样式**：
- 中文标题：3rem，Playfair Display 字体
- 英文标题：1.75rem，全大写，字距加宽
- 装饰线：60px 金色渐变线

---

### 2. 交错图文布局

**布局方式**：
```
第 1 个：左图右文
第 2 个：右图左文
第 3 个：左图右文
第 4 个：右图左文
...
```

**效果**：
- ✅ Z 字形阅读路径
- ✅ 视觉节奏感强
- ✅ 避免单调

---

### 3. 图片区域

**尺寸**：
- 桌面端：50% 宽度，600px 高度
- 平板端：100% 宽度，400px 高度
- 移动端：100% 宽度，300px 高度

**效果**：
- 淡绿色渐变遮罩
- 悬停时图片放大 5%
- 大号半透明编号（01, 02, 03...）

---

### 4. 文字区域

**排版**：
```
CORE ENTITLEMENT 01  ← 标签（金色）

康养驻留              ← 标题（深绿，2.5rem）
Wellness Residency   ← 英文标题

每年 30/60/90 天...    ← 描述（1.125rem）
30/60/90 days...     ← 英文描述

────                 ← 装饰线（金色渐变）
```

**字体**：
- 中文标题：Playfair Display, 2.5rem
- 英文标题：Lato, 1.75rem
- 标签：全大写，金色，字距加宽

---

### 5. 编号设计

**图片编号**：
```css
位置：左上角
字体：Playfair Display
大小：4rem
颜色：rgba(255, 255, 255, 0.3)
效果：半透明，文字阴影
```

**标签编号**：
```css
位置：文字区域顶部
格式：CORE ENTITLEMENT 01
颜色：金色 (#c9a962)
效果：全大写，字距加宽
```
