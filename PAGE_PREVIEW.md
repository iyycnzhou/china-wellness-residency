### 2️⃣ 动画效果

**缓慢缩放动画**：
```css
@keyframes slowZoom {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.05); }
}
/* 20 秒循环，营造呼吸感 */
```

**效果描述**：
- 图片在 20 秒内缓慢放大 5%
- 然后缓慢缩小回原样
- 循环往复，营造"呼吸"感
- Six Senses 网站经典效果

### 2. 遮罩渐变

```css
background: linear-gradient(135deg, 
  rgba(26, 74, 62, 0.7) 0%, 
  rgba(26, 74, 62, 0.5) 50%, 
  rgba(26, 74, 62, 0.6) 100%);
```

**效果**：
- 左上角较深（0.7 透明度）
- 中间较浅（0.5 透明度）
- 右下角中等（0.6 透明度）
- 营造立体感

### 3. 文字阴影

```css
text-shadow: 0 2px 10px rgba(0, 0, 0, 0.5);
```

**作用**：
- 增强文字可读性
- 营造层次感
- 避免文字与背景混在一起

## 🎩 Sophie 的设计思路

**先生**：

我按照 Six Senses 的风格进行了以下优化：

### 1. 色彩统一

所有颜色都来自 Six Senses 官网的配色：
- 从官网提取主色调
- 定义 CSS 变量
- 全局统一使用

### 2. 字体优雅

选择了经典的字体组合：
- 标题：Playfair Display（奢华感）
- 正文：Lato（现代感）

### 3. 动画细腻

- 缓慢的缩放动画（20 秒）
- 平滑的缓动函数
- 避免生硬的过渡

### 4. 图片质量

- 使用高质量摄影图片
- 避免 AI 生成的廉价感
- 色调统一协调
