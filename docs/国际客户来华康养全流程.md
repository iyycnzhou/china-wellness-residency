├─ 填写基本联系信息（姓名、邮箱、电话）
├─ 选择服务类型：

│  ├─ 🏥 医疗预约（肿瘤筛查、心脏检查、体检）
│  ├─ 🌿 中医调理（针灸、推拿、草药）
│  ├─ 🧘 康养驻留（7天/14天/30天套餐）
│  └─ 🏖️ 康复治疗（术后康复、理疗）
└─ 提交咨询表单 → 获得自动回复（10分钟）

第 3 步：初步沟通（48小时内）
├─ 客服团队收到预约请求
├─ 多语言客服（中/英/俄/阿）主动联系患者
├─ 了解详细病情（或健康需求）
│  ├─ 问：年龄、性别、既往病史
│  ├─ 问：目前症状、诊断报告
│  ├─ 问：来华时间规划、预算范围

└─ 安排首次医疗评估（视频会议）

第 4 步：方案定制（1对1定制）
├─ 客户经理+医疗顾问组成小组
├─ 分析患者病情和需求
├─ 推荐合作医院/中医机构/康养目的地
├─ 提供3个选项方案：
│  ├─ 方案A：标准流程（10天）
│  ├─ 方案B：深度治疗（14天）
│  └─ 方案C：高端定制（21天+）
└─ 提供详细报价单（含机票、签证、住宿）

第 5 步：预约确认（签署协议）
├─ 患者选择方案并支付定金（10-30%）
├─ 签署服务协议（中英双语）
├─ 获取确认函（Invoice + Service Confirmation）
├─ 提交以下材料：
│  ├─ 护照扫描件（有效期6个月以上）
│  ├─ 近期体检报告
│  ├─ 签证申请材料（如需要协助）
│  └─ 紧急联系人信息
└─ 入境材料包发送（vaccination certificate, itinerary, hotel confirmation）

第 6 步：出行准备（行前指导）
├─ 行前15天：发送《行前指南》
│  ├─ 中国天气预报（到达城市）
│  ├─ 机场通关须知
│  ├─ 医保/外汇兑换提醒
│  └─ 必备药品清单（中英对照）
├─ 行前7天：视频会议确认
│  ├─ 服务流程回顾
│  ├─ 医院/酒店地址确认
│  └─ 紧急联系人确认
└─ 行前1天：提醒短信/邮件
   ├─ 航班信息确认
   ├─ 接机安排确认
   └─ 首日行程提醒

第 7 步：抵达中国（到达日）
├─ 抵达指定机场（上海/北京/成都等）
├─ 接机服务（司机举牌等候）
├─ 护送至酒店/医院
├─ 办理入住手续
└─ 首日欢迎仪式（欢迎茶/简餐）

第 8 步：诊疗阶段（核心服务）
├─ Day 1: 初诊评估
│  ├─ 医生面诊（翻译陪同）
│  ├─ 现场检查（CT/B超/心电图等）
│  └─ 初步诊断报告
├─ Day 2-3: 治疗方案讨论
│  ├─ 多学科会诊（MDT）
│  ├─ 方案确认（手术/保守治疗/中医调理）

├─ Day 4-7: 实施治疗
│  ├─ 门诊治疗/住院手术
│  ├─ 中医配合（针灸/推拿/草药）
│  └─ 每日病情沟通会
└─ Day 8-14: 康复驻留
   ├─ 康养中心入住
   ├─ 日常康复训练
   └─ 饮食调理（营养师定制餐）

第 9 步：术后随访（延续服务）
├─ 出院后3天：电话随访
├─ 出院后7天：视频复诊
├─ 出院后30天：远程评估

```
CNWR-{YYYYMMDD}-{6位数字}
```

**示例**：`CNWR-20260317-123456`

- `CNWR` = China Wellness Residency
- `20260317` = 提交日期
- `123456` = 当日顺序号
