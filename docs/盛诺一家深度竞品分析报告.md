## 🔄 盛诺一家服务流程

### 出国看病流程（"走出去"）

```
咨询评估 → 病历整理 → 医院申请 → 签证办理 → 行程安排 → 海外就医 → 回国随访
   ↓          ↓          ↓          ↓          ↓          ↓          ↓
免费初诊   翻译整理   预约医院   医疗签证   机票住宿   治疗 + 陪同   远程复查
```

**两个阶段：**

**第一阶段：诊断与方案制定（2-4 周）**
- 患者到国外医院见主治医生
- 接受面诊和相关检查
- 多学科团队评估病情、确诊并制定个性化治疗方案
- 通常需要 1-2 周

**第二阶段：治疗实施（按需）**
- 患者和家属决定是否在国外医院实施治疗
- 盛诺一家协助安排住院、治疗、康复
- 专业医学翻译全程陪同

### 来华就医流程（"引进来"，2026 新业务）

```
在线咨询 → 签证评估 → 行程规划 → 抵达接待 → 医疗服务 → 康复疗养 → 回国随访
   ↓          ↓          ↓          ↓          ↓          ↓          ↓
多语言客服   免签/医疗签   机票 + 酒店   接机 + 入住   诊疗 + 翻译   疗养 + 康复   远程复查
```
