## 🚀 快速开始

### 步骤 1：收集图片

**Sophie 的建议**：

1. **打开 Unsplash**
   ```
   https://unsplash.com/s/photos/luxury-wellness-resort
   ```

2. **搜索关键词**
   ```
   luxury resort, wellness spa, nature landscape
   ```

3. **下载图片**
   - 选择高质量图片（至少 1920px 宽）
   - 保存到临时文件夹

---

### 步骤 2：整理分类

```bash
mkdir -p ~/Downloads/wellness-images/{banner,services,icons,partners}
```

---

### 步骤 3：优化压缩

1. **打开 TinyPNG**
   ```
   https://tinypng.com/
   ```

2. **拖拽图片**
   - 批量上传
   - 自动压缩
   - 下载优化后的图片

---

### 步骤 4：放入项目

```bash
# 复制图片到项目
cp ~/Downloads/wellness-images/banner/* /mnt/d/openclaw/workspace/china-wellness-residency/public/images/banner/
cp ~/Downloads/wellness-images/services/* /mnt/d/openclaw/workspace/china-wellness-residency/public/images/services/
```

## 💡 图片使用技巧

### 1. Banner 图片

**CSS 使用**：
```css
.hero-banner {
  background-image: url('/images/banner/hero-1.jpg');
  background-size: cover;
  background-position: center;
  height: 100vh;
}
```

**Vue 组件使用**：
```vue
<template>
  <div class="hero-banner">
    <img src="/images/banner/hero-1.jpg" alt="豪华度假村" />
  </div>
</template>
```

---

### 2. 响应式图片

```html
<picture>
  <source media="(max-width: 768px)" srcset="/images/banner/hero-mobile.jpg">
  <source media="(min-width: 769px)" srcset="/images/banner/hero-desktop.jpg">
  <img src="/images/banner/hero-desktop.jpg" alt="度假村" />
</picture>
```

---

### 3. 懒加载

```vue
<template>
  <img 
    src="/images/services/hospital.jpg" 
    alt="医院"
    loading="lazy"
  />
</template>
```

## ⚠️ 注意事项

### 版权说明

✅ **可以使用**：
- Unsplash、Pexels、Pixabay 的免费图片
- 标注了 CC0、可商用的图片
- 自己拍摄的图片

❌ **不能使用**：
- 有明确版权标识的图片
- Google 图片搜索直接保存的图片
- 未授权的商业图片

---

### 图片质量

✅ **推荐**：
- 分辨率至少 1920px 宽
- 文件格式 JPG 或 WebP
- 文件大小 < 500KB（压缩后）

❌ **避免**：
- 模糊不清的图片
- 有明显水印
- 文件过大（> 1MB）
