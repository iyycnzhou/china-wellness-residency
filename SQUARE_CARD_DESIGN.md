## 🎨 设计细节

### 1. 独立方形区域

**图片区域**：
```css
.benefit-showcase-image {
  aspect-ratio: 1 / 1;  /* 强制正方形 */
  border-right: 3px solid var(--secondary);  /* 金色分隔 */
}
```

**文字区域**：
```css
.benefit-showcase-content {
  border-left: 3px solid var(--sand);  /* 淡色分隔 */
  padding: var(--spacing-lg);  /* 24px 内边距 */
}
```

### 2. 分隔线设计

**视觉效果**：
```
┌──────────────┬────────────────┐
│              │                │
│              │ ← 3px 金色边框  │
│    图片区    │                │
│              │    文字区      │
│              │                │
│              │ → 3px 沙色边框  │
│              │                │
└──────────────┴────────────────┘
```

**颜色**：
- 图片右侧：金色 `#c9a962`（醒目）
- 文字左侧：沙色 `#f5f1e8`（柔和）

### 3. 文字排版

**层次结构**：
```
核心权益 01  ← 标签（金色，0.6875rem）
──────────
康养驻留     ← 中文标题（1.5rem）
WELLNESS... ← 英文标题（1.125rem）
──────────
每年可享...  ← 中文描述（0.875rem）
Enjoy...    ← 英文描述（0.8125rem）
──────────
────        ← 装饰线（40px 金色）
```

优点：

## ✅ 修改总结

### CSS 修改

1. **卡片尺寸**
   ```css
   min-height: 280px;  /* 降低高度 */
   min-width: 600px;   /* 增加宽度 */
   aspect-ratio: auto; /* 取消整体比例 */
   ```

2. **图片区域**
   ```css
   aspect-ratio: 1 / 1;  /* 强制正方形 */
   border-right: 3px solid var(--secondary);
   ```

3. **文字区域**
   ```css
   border-left: 3px solid var(--sand);
   padding: var(--spacing-lg);
   ```

4. **文字大小**
   ```css
   .title-cn { font-size: 1.5rem; }
   .title-en { font-size: 1.125rem; }
   .desc-cn { font-size: 0.875rem; }
   .desc-en { font-size: 0.8125rem; }
   ```
